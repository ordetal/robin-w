# Сайт "Robin. W"

Проект создан по данному [**макету**](https://www.figma.com/design/OrtjNW9auxUNJPtxg08sik/Robin.-W?node-id=9-141&p=f&t=8Q3fv0PC3KqGxFs8-0) в **Figma** с использованием препроцессора стилей ***SCSS***.

## Стек

- HTML
- SCSS / CSS

Ниже представлена структура сайта, и все указанные пути в файлах используют её.

## Как запустить

Сохраните проект с помощью команды `git clone https://github.com/ordetal/robin-w.git` (или любым другим известным вам способом) и откройте файл **_index.html_**.

## Как работать

Для того чтобы **scss** скомпилировался в **css**, я использовал расширение в VS Code **"Live Sass Compiler"**, Вы же можете использовать любой другой удобный для вас инструмент сборки.
